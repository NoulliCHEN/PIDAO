After downloading the file, you need to create an empty folder in this path named 'figure'.

for the comparison of the strongly convex objective (quadratic loss): run continuous_time_comparison_SC.py;

for the comparison between EPIDAO and PIDAO: run continuous_time_EPID_vs_PID.py;

for the comparison of different k_i: run ki_effect.py;

for the comparison of the Rastrigin/Rosenbrock/Six hump camel loss: run each Python file aligning the corresponding name.
